🕒 Minimalist Glassmorphism Stopwatch
A sleek, responsive, and modern digital stopwatch built for precision and style.

🔗 Live Demo
Check out the live application here: https://danithk-pradeep.github.io/Minimalist-Glassmorphism-Stopwatch/

📝 Project Description
This project is a high-performance web-based stopwatch application. It combines a clean "Glassmorphism" UI with robust functionality, allowing users to track time accurately down to the millisecond.With its intuitive Start, Stop, and Reset controls, it is perfect for timing workouts, study sessions, or any task requiring focus. The application is built entirely using Vanilla JavaScript, ensuring it remains lightweight and fast without any external dependencies.

✨ Key Features
Precision Timing: Accurately tracks hours, minutes, seconds, and milliseconds.
Modern UI: Features a professional Glassmorphism effect with a blurred background.
Fully Responsive: Optimized for a seamless experience on mobile, tablet, and desktop screens.
Interactive Controls: Smoothly transition between Start, Stop, and Reset states.

🛠️ Technologies Used
HTML5: Structured for accessibility and semantic clarity.
CSS3: Utilizes Flexbox and backdrop-filter for the modern glass effect.
JavaScript (ES6): Handles the core logic and real-time DOM updates.
